================================================================================
_simpson_weights(N) -> ndarray   [static method]
================================================================================

PURPOSE
-------
Generates the quadrature weights for Simpson's 1/3 rule over N equally-spaced
points. These weights are used to approximate the continuous Fourier integral
in the Carr-Madan method as a discrete sum, improving accuracy over the
simple trapezoidal rule.

INPUTS
------
  N : int - Number of quadrature points (typically a power of 2, e.g. 4096,
            to enable efficient FFT computation).

OUTPUTS
-------
  ndarray of shape (N,) - Simpson weights, already divided by 3.

MATHEMATICS
-----------
Simpson's 1/3 rule approximates a definite integral using quadratic
interpolation between successive groups of three points:

    integral from a to b of f(x) dx
      ~ (h/3) * [f(x_0) + 4*f(x_1) + 2*f(x_2) + 4*f(x_3) + ... + f(x_N-1)]

where h is the spacing between points.

The weight pattern is: 1, 4, 2, 4, 2, 4, 2, ..., 4, 1
divided by 3.

For N points, the error is O(h^4), compared to O(h^2) for the trapezoidal
rule. This means Simpson's rule achieves much higher accuracy for smooth
integrands with the same number of points, which is important for the
Carr-Madan FFT integral where accuracy depends on the numerical quadrature.

EDGE CASE:
When N is even, the last weight is set to 1.0 (matching the standard
Simpson pattern endpoint). The method always sets w[0] = 1, odd indices
to 4, even indices to 2, then adjusts the last element if N is even.

CONNECTION TO THE FFT PRICER:
In the Carr-Madan method, the call price integral is:

    C(K) = (exp(-alpha*k)/pi) * integral_0^inf Re[exp(-iuk) * Psi(u)] du

This integral is discretized as:

    C(K) ~ (exp(-alpha*k)/pi) * sum_{j=0}^{N-1} exp(-i*u_j*k) * Psi(u_j) * eta * w_j

where eta is the integration spacing in u-space and w_j are the Simpson
weights. The factor eta*w_j replaces the simple du increment, providing
higher-order accuracy.

PSEUDOCODE
----------
  def _simpson_weights(N):
      w = empty array of size N
      w[0] = 1
      w[odd indices] = 4
      w[even indices >= 2] = 2
      if N is even:
          w[N-1] = 1
      return w / 3
